# herd-store-duckdb

**Herd storage adapter — local DuckDB + MotherDuck.**

Persists agent sessions, decision records, handoffs, and operational data.
Works with local DuckDB for development and MotherDuck for shared/production state.

Part of [The Herd](https://github.com/dbt-conceptual/herd-core) ecosystem.

## Status

Pre-release — package name reserved.

## Install

```bash
pip install herd-core herd-store-duckdb
```
